# 项目结构说明

本项目所有文件和环境都位于项目根目录内，完全自包含。

## 📁 目录结构

```
Ourmind/
├── ourmind_env/          # Python虚拟环境（所有Python依赖）
├── node_modules/         # Node.js依赖包
├── uploads/              # 上传文件存储目录
├── backend/              # 后端代码
│   ├── main.py          # FastAPI后端服务器
│   └── __pycache__/     # Python缓存文件
├── src/                  # 前端代码
│   ├── components/      # React组件
│   ├── App.jsx          # 主应用
│   └── ...
├── .env                  # 环境变量配置（API Key等）
├── package.json         # 前端依赖配置
├── requirements.txt     # 后端依赖配置
└── quick_start.sh       # 一键启动脚本
```

## 🔧 环境说明

### Python环境
- **位置**: `ourmind_env/` (项目根目录内)
- **包含**: 所有Python依赖包（FastAPI、OpenAI、PyPDF2等）
- **激活**: `source ourmind_env/bin/activate`

### Node.js环境
- **位置**: `node_modules/` (项目根目录内)
- **包含**: 所有前端依赖（React、Vite、react-pdf等）

### 上传文件
- **位置**: `uploads/` (项目根目录内)
- **用途**: 存储用户上传的PDF、DOCX、TXT文件

## 📝 配置文件

- `.env`: DeepSeek/OpenAI API Key配置
- `package.json`: 前端依赖和脚本
- `requirements.txt`: 后端Python依赖
- `vite.config.js`: Vite构建配置
- `tailwind.config.js`: Tailwind CSS配置

## 🚀 启动方式

所有启动脚本都使用相对路径，确保在项目目录内运行：

```bash
./quick_start.sh
```

## ✅ 自包含特性

- ✅ Python虚拟环境在项目内
- ✅ Node.js依赖在项目内
- ✅ 所有配置文件使用相对路径
- ✅ 上传文件存储在项目内
- ✅ 无需外部依赖或全局安装

## 📦 项目大小

- Python虚拟环境: ~200MB
- Node.js依赖: ~150MB
- 项目代码: ~1MB
- **总计**: ~350MB

## 🔒 安全提示

以下文件/目录不应提交到版本控制：
- `.env` (包含API密钥)
- `ourmind_env/` (虚拟环境)
- `node_modules/` (依赖包)
- `uploads/` (用户上传文件)
- `__pycache__/` (Python缓存)

这些已在 `.gitignore` 中配置。
